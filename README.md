###wheninrome###

wheninrome is JSP taglib wrapper on the Rome Atom/RSS syndication tools.


Example JSP file:
    webapp/rsstest.jsp

To build, run
    bin/build 
to create 
    build/wheninrome.jar
that you install in your webapp's classpath.

Requirements

- jdom-1.0.jar
- jsp-api-2.0.jar
- jstl.jar
- log4j-1.2.15.jar
- rome-0.9.jar
- servlet-api-2.4.jar
